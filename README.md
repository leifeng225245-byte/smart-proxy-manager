# 智能翻墙管理系统

## 🎯 项目简介

智能翻墙管理系统是一个专为Ubuntu用户设计的网络管理工具，完美解决了"关闭翻墙后无法直连上网"的问题。

## 🚀 核心功能

- ✅ **一键开关** - 单条命令完成翻墙开启/关闭
- ✅ **智能管理** - 自动管理环境变量和系统代理
- ✅ **状态记忆** - 重启后自动恢复上次状态
- ✅ **安全备份** - 自动备份配置和状态
- ✅ **完整监控** - 详细的状态显示和日志记录

## 📦 快速开始

### 安装步骤

1. **下载脚本**
   ```bash
   wget https://raw.githubusercontent.com/your-username/smart-proxy-manager/main/mihomo-manager.sh
   chmod +x mihomo-manager.sh
   ```

2. **配置mihomo**
   ```bash
   mkdir -p ~/.config/mihomo
   # 创建配置文件（参考文档中的模板）
   ```

3. **创建系统服务**
   ```bash
   sudo cp mihomo.service /etc/systemd/system/
   sudo systemctl enable mihomo
   ```

### 基本使用

```bash
# 开启翻墙
./mihomo-manager.sh on

# 关闭翻墙
./mihomo-manager.sh off

# 查看状态
./mihomo-manager.sh status
```

## 📋 系统要求

- **操作系统**: Ubuntu 20.04+
- **桌面环境**: GNOME
- **代理服务**: mihomo (Clash Meta)
