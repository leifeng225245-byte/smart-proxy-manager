# 翻墙开关使用指南

## 🎯 最简单的使用方法

### 🔥 每天就这几个命令
```bash
# 开翻墙
./mihomo-manager.sh on

# 关翻墙
./mihomo-manager.sh off

# 看状态
./mihomo-manager.sh status
```

### 📱 什么时候用
```
开机后 → 自动翻墙，不用管
要上百度 → ./mihomo-manager.sh off
要上谷歌 → ./mihomo-manager.sh on
不确定状态 → ./mihomo-manager.sh status
```

---

## 🎮 命令说明

| 命令 | 什么时候用 | 效果 |
|------|------------|------|
| `on` | 要上国外网站 | 开启翻墙 |
| `off` | 要上国内网站 | 关闭翻墙，直连 |
| `status` | 不知道当前状态 | 显示翻墙状态 |
| `test` | 网络有问题 | 测试网络 |
| `backup` | 怕改坏了 | 备份配置 |
| `help` | 忘记命令 | 显示帮助 |

---

## 🚀 实际使用例子

### 场景1：日常使用
```bash
# 早上开机，自动翻墙，直接用
# 中午要查资料，上百度
./mihomo-manager.sh off

# 下午要上谷歌
./mihomo-manager.sh on
```

### 场景2：网络有问题
```bash
# 先看看什么状态
./mihomo-manager.sh status

# 测试网络
./mihomo-manager.sh test

# 重新开关
./mihomo-manager.sh off
./mihomo-manager.sh on
```

### 场景3：不确定状态
```bash
# 看状态
./mihomo-manager.sh status

# 如果显示"已停止"，就开翻墙
./mihomo-manager.sh on

# 如果显示"运行中"，就关翻墙
./mihomo-manager.sh off
```

## ⚠️ 注意事项

### 🔒 安全第一
- 每次操作前自动备份
- 出问题可以恢复
- 不用担心改坏

### 🐛 常见问题

#### 问题1：命令不管用
```bash
# 检查权限
chmod +x ~/mihomo-manager.sh

# 重新运行
./mihomo-manager.sh on
```

#### 问题2：开了翻墙但上不了网
```bash
# 检查状态
./mihomo-manager.sh status

# 重新开一次
./mihomo-manager.sh off
./mihomo-manager.sh on
```

#### 问题3：关了翻墙但上不了网
```bash
# 检查状态
./mihomo-manager.sh status

# 手动清除变量
unset HTTP_PROXY HTTPS_PROXY ALL_PROXY

# 重新开关
./mihomo-manager.sh off
```

## 🎯 核心优势

### ✨ 为什么好用
- **一条命令**：不用记复杂的操作
- **自动备份**：改坏了能恢复
- **状态记忆**：重启后保持上次选择
- **中文提示**：看得懂错误信息
- **颜色显示**：一眼看出状态

### 🔄 智能功能
```
开机 → 自动翻墙（如果上次是开的）
关翻墙 → 自动清理环境变量
开翻墙 → 自动设置环境变量
每次操作 → 自动备份配置
```
